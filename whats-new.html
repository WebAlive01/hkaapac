Three Powerful New Features

PMWeb version 8.0 features three brand new features, as well as a number of enhancements to existing features and bug fixes.

Adaptive Forms — Configurable dynamic forms that respond to user input and workflow state, enabling more intelligent data capture across project records.
Power BI Reporting integrated into PMWeb reporting — Native Power BI dashboards embedded directly within PMWeb's reporting module, providing richer visual analytics without leaving the platform.
Multi Factor Authentication — Enhanced security for all PMWeb deployments, supporting MFA across browser and mobile access.

For over a decade, the world's most demanding construction professionals have relied on PMWeb to provide unrivalled power, flexibility, and ease of use, in a single, integrated, online platform. To that feature list, PMWeb has now added complete portability on every device.

PMWeb 7 is optimised to load and run faster on devices large and small, even when managing the huge volume of data that modern projects generate. That applies to every feature in every PMWeb module, not a watered-down subset of field tools — and the new responsive design engine automatically adjusts the entire application to fit the screen you're viewing. Best of all, every feature of PMWeb is available without installing, maintaining, or worrying about, a single app. All you need is a browser and a password to immediately start working with PMWeb 7.

Responsive design — PMWeb 7 automatically detects the device you are using and the orientation of the screen and adapts itself to make best use of the available space.
All new interface — Every page of every record type has been recoded, refreshed, and refined. Screens automatically adapt, vertical line editing on narrow devices, faster performance across the platform.
Redesigned home page — 100% redesigned Project Centre tab, full screen interactive map with spreadsheet overlay, and customisable background image.
Control panel — The menu has been replaced by a responsive, collapsible control panel. Menus open in easy to read, space saving, overlay mode with logically regrouped and sorted choices.
Drag and drop — Drag and drop metaphor supported on every device and platform. Responsive dialogs adapt to device form factors without sacrificing function.
Colours and personalisation — Create your own interface colour scheme. Add a personalised image to the login page. Redesigned My Settings makes personalising PMWeb faster and easier.

Full Version 5.2 Details

PMWeb Version 5.1 introduces two major features — the PMWeb 3D Viewer and PMWeb University — as well as a large number of enhancements and issue resolutions.

PMWeb 3D Viewer — The PMWeb 3D Viewer can be used to view and interact with many types of 3D file formats. Use the 3D Viewer to inspect models, rotate them, explode them, and take measurements. The 3D Viewer can be opened from any Attachments tab, the Document Manager, and from a Model Manager record. Use the image Snapshot feature to save images as attachments to a PMWeb record, create a new PMWeb record (an RFI, for example) with the image attached, create an email with the image attached, or print the image. The PMWeb 3D Viewer uses your Autodesk© Forge account to upload and directly work with 3D models, online, inside of PMWeb. The 3D Viewer is licensed as part of the optional BIM module.
PMWeb University — PMWeb University is a comprehensive suite of online instructional videos written and recorded by PMWeb subject matter experts. As soon as you have licensed PMWeb University, users can click a button at the top of the home page to access the full range of training content. They can view each video as often as they wish and can complete topics at their own pace.
Read more about PMWeb's powerful BIM features

PMWeb Version 5.0, released in November 2016, was a major software release — transitioning the leading enterprise-level project management software for the construction industry to new prominence as the world's most robust and user-friendly project management platform.

Capital project owners, facility managers, program and construction managers and contractors in industries such as Aviation/Ports, Construction/Design, Education, Energy/Resources, Government, Healthcare, Infrastructure, Retail/Commercial, among others, all benefit from the system wide improvements in Version 5.0. The feedback and requests from PMWeb's broad global client base were a key driver in the development plan for Version 5.0.

The release of PMWeb Version 5.0 stays true to the company's on-going pledge to continually innovate as technological leaders in the industry. The editors of Constructech agreed: "PMWeb not only knows what the construction industry needs, but also delivers a comprehensive, integrated solution. With a focus on client services and R&D, the company is a good one to do business with."

New Layout and Architecture — PMWeb's Version 5.0 features a brand-new look and feel. All controls throughout the program were upgraded to be lighter on the page, resulting in faster load and save times. The icon set was refreshed and a new "Metro" skin added.
Reinvented Scheduling — The Scheduling Module was reinvented from the ground up in Version 5. The interface was overhauled and many new features were added that make the Scheduling Module much more powerful and easier to use.
Advanced Multi-Currency — PMWeb has long had multi-currency by project, but now that capability has been expanded down to the line item level. Users can use multiple currencies among the lines of a single record — for example, a Purchase Order with lines in US Dollars, Euros, and Yuan.
BIM Revit® Add-in — PMWeb can now integrate directly with BIM models using the Revit® add-in. This is used to automatically create PMWeb records from the BIM model, including assets (floors, spaces, equipment records) as well as RFIs, Submittals, and Work Orders.
PBS and Project Explorer — A new PBS field added to Programs, Projects and Initiatives. The PBS page in the Portfolio module provides an unlimited user-defined tree structure — drag and drop your projects onto any node to classify them.
New Collaboration Tools — A new "Collaborate" tab available in most record types. Users can invite others to review and contribute to records on the fly. Collaboration actions are logged and time and date stamped by user to automatically create detailed "conversations".
Multi-Project Records — In the cost management module, Budget Requests and Journal Entries can now be made as multi-project records. Each line can be assigned to a different project.
Enhanced Scoring — The scoring features have been greatly enhanced for Version 5.0. For every Scoring question, you can now define the data type of the answer.
